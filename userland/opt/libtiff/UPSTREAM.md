The ArmOS import keeps the core `libtiff` sources, public headers, private
headers needed for a static build, and upstream documentation/license files.
The ArmOS configuration enables baseline TIFF codecs plus JPEG, LZW, PackBits,
CCITT, ThunderScan, NeXT, and Deflate/ZIP support.
